# Academic — Format Markdown/Word-style

**Module:** H2 — Platform
**Loại:** ⚪ Tùy chọn
**Mục đích:** Format bài academic dạng Markdown/Word-style (khác `facebook.md` — ở đây GIỮ NGUYÊN heading/cấu trúc, không xoá).

---

## Nguyên tắc cốt lõi

Ngược hoàn toàn với `facebook.md`:
- GIỮ NGUYÊN heading `#`, `##`, `###` theo cấu trúc IMRaD hoặc mục lục báo cáo
- GIỮ NGUYÊN numbering mục (1., 1.1, 1.2...)
- Bullet/numbered list dùng bình thường cho danh sách specs, bước, so sánh
- Có phần Tài liệu tham khảo/References ở cuối bài (KHÔNG bỏ như facebook.md)

## Cấu trúc file

```
# [Tiêu đề bài]

## Tóm tắt (Abstract)
[150-250 từ]

**Từ khóa (Keywords):** [3-5 từ khóa]

## 1. Giới thiệu
...

## 2. [Mục tiếp theo theo cấu trúc đã chọn]
...

## Tài liệu tham khảo
[Danh mục theo citation style đã chọn — xem review/citation-check.md]
```

## Quy ước bảng/hình

| Ngôn ngữ | Caption bảng | Caption hình |
|---|---|---|
| Tiếng Việt | "Bảng 1. [Mô tả]" phía trên bảng | "Hình 1. [Mô tả]" phía dưới hình |
| Tiếng Anh | "Table 1. [Description]" above table | "Figure 1. [Description]" below figure |

## Checklist

- [ ] Heading/numbering giữ nguyên theo cấu trúc academic
- [ ] Có Tóm tắt/Abstract + Từ khóa/Keywords
- [ ] Tài liệu tham khảo đầy đủ, đúng citation style
- [ ] Caption bảng/hình đúng vị trí và ngôn ngữ
- [ ] Nội dung giữ nguyên 100% so với bản editorial đã duyệt (chỉ format)
