# Lật Góc — Reframe

**Nhân viên:** reframe.md
**Ban:** Biên tập (editorial/)
**Chức năng:** Đặt tên hiện tượng, đảo kỳ vọng, giải thích bằng cấu trúc song song.

---

## 1. Đặt tên hiện tượng (Concept Naming)

**Công thức:**
```
Mô tả hiện tượng bằng ví dụ cụ thể
    ↓
Đặt tên ngắn gọn, đáng nhớ (ngoặc kép)
    ↓
Dùng tên đó xuyên suốt bài — nó trở thành concept
```

**Ví dụ:**
```
✅ "khoán trắng tư duy" — nhân viên copy AI 100%, không kiểm tra
✅ "zombie công sở" — năng suất cao, báo cáo đẹp, bên trong rỗng
✅ "nồi áp suất" — áp lực từ mọi phía trong nghiên cứu khoa học
```

**Cấm:** Tên quá dài (>4 từ) hoặc quá trừu tượng. Tên tốt = hình ảnh + cảm xúc.

### 1.1 Hệ thống khái niệm (Concept System)

Khi bài cần >1 concept, các concept phải liên kết logic thành hệ thống, dùng chung 1 "ngôn ngữ" hoặc 1 phép ẩn dụ xuyên suốt.

**Công thức:**
```
Concept A (quen thuộc) → Concept B (lật góc) → Concept C (nâng tầm)
Tất cả dùng chung 1 hệ thống ngôn ngữ.
```

**Ví dụ:**
```
✅ "phép cộng" → "phép nhân" → "phép lũy thừa"
    Tất cả dùng ngôn ngữ toán học, xoay quanh "cơ số phải lớn hơn 1"

❌ "phép nhân" + "cánh cửa" + "tảng băng"
    Ba ẩn dụ không liên kết → rời rạc, không thành hệ thống
```

**Cấm:** >3 concept/bài. Hệ thống quá phức tạp sẽ mất tập trung.

## 2. Đảo kỳ vọng (Paradox Flip)

**Công thức:**
```
[Điều mọi người nghĩ] + "nhưng thực ra" + [điều ngược lại]
```

**Ví dụ:**
```
✅ "Bi quan luôn nghe thông minh hơn lạc quan —
    vì lạc quan nghe như selling, bi quan nghe như đang giúp bạn."

✅ "Giàu lên và giữ được giàu là hai việc khác nhau,
    đòi hỏi hai bộ kỹ năng khác nhau."
```

**Cấm:** Đảo chỉ để gây sốc. Paradox phải chứa truth.

---

## 3. Cấu trúc song song (Parallel Analogy)

**Công thức:**
```
[Lĩnh vực quen thuộc] biết bao nhiêu người [X].
[Lĩnh vực của bạn] biết bao nhiêu người [Y].
→ X và Y song song → insight hiện ra tự nhiên.
```

**Ví dụ:**
```
✅ "Thời tiền công nghiệp, công việc giúp bạn khỏe.
    Giờ muốn khỏe, phải chủ động tập gym.
    Tương tự: AI sẽ khiến viết như gym — chỉ ai chọn mới giỏi."
```

**Cấm:** Song song gượng ép. Nếu phải giải thích sự song song thì nó không hiệu quả.

## Checklist

- [ ] Concept naming ngắn gọn, đáng nhớ (≤ 4 từ)?
- [ ] Paradox chứa truth, không chỉ gây sốc?
- [ ] Parallel analogy tự nhiên, không cần giải thích?
